## Hacker News Post

**Title**: Show HN: Architecture diagram builder in Angular with 47+ cloud/AI components

**Text**:
```
I built an open-source architecture diagram builder using Angular and Konva.js.

It includes 47+ pre-built components for AWS, Azure, GCP, and AI/ML tools (LangChain, vector databases, LLMs, etc.). Features drag-and-drop, nested grouping, custom components with image upload, and JSON export.

Live demo: https://samba425.github.io/konva-architecture-canvas/

Started this because I needed a quick way to visualize RAG architectures and cloud infrastructure for documentation. Turns out it's useful for system design interviews and technical presentations too.

Built with Angular 21 and Konva.js for high-performance canvas rendering. MIT licensed.

The tool addresses some pain points I had with existing solutions:

1. Integration: It's an npm package, not a separate app
2. Components: Pre-built tech stack components (no icon hunting)
3. Export: JSON format for version control
4. Keyboard-driven: Full shortcuts (Cmd+G, Cmd+Z, etc.)

Some interesting technical decisions:
- Chose canvas (Konva) over SVG (D3) for better performance
- Implemented command pattern for undo/redo
- Virtual scrolling for component sidebar
- Auto-save to localStorage

Current limitations:
- No real-time collaboration (yet)
- Limited to web (no offline desktop app)
- Bundle size ~680KB

Roadmap:
- Export to Terraform/CDK
- Custom component creator
- AI-powered auto-layout
- VS Code extension

Would love feedback on what features to prioritize!

GitHub: https://github.com/samba425/konva-architecture-canvas
npm: npm install konva-architecture-canvas
```
